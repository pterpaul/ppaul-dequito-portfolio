- cybersecurity-focused growth
- deployed project work
- technical stack and certifications

## Overview

The website highlights my professional work in:

- IT Support L0-L3
- Systems Administration
- Network and Infrastructure Support
- Cybersecurity Analysis
- Incident Response
- Web-based business systems

The goal of the project is not only to present information, but to make the portfolio feel more interactive, modern, and data-aware through dashboards, carousels, and conversational portfolio access.

## Key Features

- Responsive portfolio landing page
- Structured About, Experience, Skills, Projects, Reviews, Gallery, and Contact sections
- Swiper-powered carousels for projects, reviews, and gallery
- Chart.js dashboard for review analytics
- Custom chatbot for portfolio questions and guided responses
- Privacy-aware chatbot handling for sensitive personal questions
- Light and dark theme support

## Chatbot Features

The portfolio chatbot can respond to questions about:

- experience and current work
- projects and case studies
- technical tools and stack
- certifications
- contact details
- privacy-safe public location

It also includes conversational handling for:

- greetings
- thanks
- humor
- casual reactions like `wow`, `nice`, and `amazing`
- inappropriate language filtering
- privacy boundaries for personal questions

## Running Locally

You can open `index.html` directly in a browser, but using a local server is recommended.

Example:

```bash
npx serve .
```

Then open the local address provided by the server.
